# Investment-Portfolio-Performance-Analysis-Python
Python analysis of a 5-stock diversified portfolio vs the S&amp;P 500 benchmark. Calculates annualized return, volatility, Sharpe Ratio, and max drawdown. Includes correlation heatmap, monthly returns heatmap, and risk vs return visualization. Portfolio outperformed the S&amp;P 500 (Sharpe 0.61 vs 0.55).
